# AStockEveryDay — A股每日涨停预测

> AI 驱动的 A 股涨停预测站，基于公开市场数据多维交叉筛选，每日晚间自动生成次日预测。

## 项目结构

```
AStockEveryDay/
├── index.html            ← 前端 SPA（Tailwind + 纯 JS）
├── data/
│   ├── predictions.json  ← 预测索引
│   └── YYYYMMDD.json     ← 单日预测详情
└── DESIGN.md
```

纯静态站点，零构建步骤，任意静态服务器即可运行：

```bash
python -m http.server 8080
```

## 数据来源

行情、资金流向、龙虎榜、板块热度等数据均来自腾讯自选股公开行情接口。

## 回测数据

### 数据文件

回测结果存储在 `backtest_results.json`，前端回测面板自动加载展示。

每条记录包含：预测日期、股票代码/名称/排名、入场价（预测日开盘价）、出场价（截止日收盘价）、收益率、持仓天数等。

### 回测规则

1. **入场价**：预测日当日开盘价（前复权）
2. **出场价**：截止日收盘价（前复权）
3. **时间窗口**：
   - **近 10 个交易日内的预测**：从预测日的下一个交易日起追踪至当前最新交易日收盘，计算至今的累计涨幅
   - **超过 10 个交易日的预测**：只追踪预测日后 10 个交易日，以第 10 个交易日收盘价作为出场价，不再追加后续数据

### 更新方式

每次更新回测数据时，执行以下逻辑：

1. 遍历 `data/predictions.json` 中所有历史预测的股票
2. **确保最新一期预测已纳入**：前一日生成的预测（即最新交易日）的 5 只股票必须出现在回测数据中，即使仅持仓 1 个交易日也要记录当日表现
3. 对于每个预测日期，计算其距今**交易日数**
4. 若 ≤10 个交易日：查询预测日开盘价 → 最新收盘价，计算收益；**此前已存在的近 10 日内的回测记录也需同步更新**，将出场价刷新为当日最新收盘价
5. 若 >10 个交易日：查询预测日开盘价 → 第 10 个交易日收盘价，封顶不再更新
6. 写入 `backtest_results.json`，按预测日期倒序排列
7. 前端页面刷新后自动展示最新回测结果
