# 🚀 AStockEveryDay — A股每日涨停预测

> 基于热门股、资金流向、龙虎榜、热门概念多维数据交叉筛选，AI 驱动的 A 股涨停预测站。每日晚间自动选出次日最有可能涨停的 5 只个股，含竞价预期与上车条件。

## ✨ 核心特性

- **多维数据筛选** — 综合分析热门股、主力资金流向、龙虎榜、热门概念四大维度
- **每日自动更新** — 自动抓取当日收盘数据，晚间生成次日预测
- **5 强推荐** — 每期选出 5 只最具涨停潜力的个股，附综合评分
- **竞价作战指南** — 每只票含竞价预期区间、量能要求、具体上车条件
- **零依赖前端** — 单页 HTML + JSON 数据文件，零构建步骤，即开即用
- **Claude 设计** — 暖色调编辑风界面，衬线标题 + 环状阴影，金融版文艺沙龙

### 数据文件说明

**`data/predictions.json`** — 索引文件，首次加载时读取。记录每期预测的摘要信息：
```json
{
  "id": "20260616",
  "date": "2026-06-16",
  "weekday": "周二",
  "generated": "06-15",
  "title": "硬科技+小金属双主线 · MLCC/PCB/CPO/锗镓共振",
  "summary": "...",
  "topStocks": ["东山精密","风华高科","生益科技","云南锗业","太辰光"],
  "topCodes": ["002384","000636","600183","002428","300570"]
}
```

**`data/YYYYMMDD.json`** — 详情文件，用户点击某天预测时按需加载。包含：
- 市场快照（上证 / 创业板 / 深成指 / 涨停家数 / 最强板块）
- 当日热门概念 TOP10 + 5 日最强概念 TOP10
- 5 强个股详细分析（评分 / 核心逻辑 / 竞价预期 / 上车条件 / 止损线）
- 竞价作战时间线（9:15-10:00 分段指引）
- 风险提示清单

## 🤖 自动化流程

每日晚间 22:00 自动执行以下流程：

```
1. 抓取当日收盘数据（腾讯自选股接口）
   ├── 市场指数快照（沪/深/创）
   ├── 热门概念板块 TOP20
   ├── 主力资金流向 TOP30
   └── 龙虎榜数据（机构榜/游资榜）
2. 多维交叉筛选 → 5 强个股
3. 生成 JSON 数据文件（YYYYMMDD.json）
4. 更新预测索引（predictions.json）
5. 推送至 GitHub 仓库
```

## 🏗️ 架构

```
用户浏览器
    │  HTTP GET
    ▼
Static Server (GitHub Pages / CloudStudio / 任意静态托管)
    │
    ├── index.html ← Tailwind CDN + 纯 JS SPA
    │       │
    │       ├── init() → fetch predictions.json
    │       ├── route() → 根据 URL hash 渲染
    │       └── loadPrediction(id) → fetch YYYYMMDD.json
    │
    └── data/  ← JSON 数据层
```

- **前端**：单文件 SPA，零框架，全部渲染逻辑内置在 `index.html` 中
- **路由**：基于 `location.hash`（`#home` / `#0616`）
- **样式**：Tailwind CSS CDN + 内联自定义样式
- **数据**：纯 JSON 文件，无需后端 / 数据库

## 🎨 设计

暖色调编辑风设计语言，灵感源自 Claude / Anthropic 设计系统：

- **纸张质感** — 羊皮纸底色 `#f5f4ed`，远离冰冷屏幕感
- **陶土色品牌** — 赤陶 `#c96442` 作为品牌强调色，土质、刻意不科技
- **衬线权威** — Georgia 用于标题，体现分析分量
- **无衬线实用** — 系统无衬线字体用于 UI 文本和数据
- **A 股惯例** — 红涨绿跌，符合中文市场阅读习惯
- **环状阴影** — `0px 0px 0px 1px` 边框阴影代替传统投影

详见 [`DESIGN.md`](./DESIGN.md)。
